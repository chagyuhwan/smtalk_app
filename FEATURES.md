# 구현된 기능 리스트

## 📱 앱 개요
React Native + Expo 기반의 랜덤 채팅 앱 (별톡)

---

## 🔐 1. 인증 및 사용자 관리

### 인증 기능
- ✅ 전화번호 인증 (Firebase Authentication)
- ✅ SMS 인증 코드 발송 및 검증
- ✅ 자동 로그인 상태 유지
- ✅ 로그아웃 기능
- ✅ 인증 상태 실시간 감지 및 화면 전환

### 사용자 프로필
- ✅ 이름 설정 및 수정 (최대 20자)
- ✅ 성별 선택 (남/여)
- ✅ 나이 입력 (20세 이상 80세 이하)
- ✅ 프로필 사진 업로드 (갤러리/카메라)
- ✅ 자기소개 (bio) 작성
- ✅ BDSM 성향 설정 (지배/복종/스위치/없음)
- ✅ 지역 설정 (전국 17개 시도)
- ✅ 프로필 정보 Firestore 실시간 동기화

---

## 💬 2. 채팅 기능

### 채팅방 관리
- ✅ 1:1 채팅방 생성
- ✅ 채팅방 목록 조회 (최신 메시지 순 정렬)
- ✅ 채팅방 삭제 (스와이프 제스처)
- ✅ 읽지 않은 메시지 개수 표시
- ✅ 채팅방 실시간 업데이트

### 메시지 기능
- ✅ 텍스트 메시지 전송
- ✅ 실시간 메시지 수신 (Firestore 실시간 구독)
- ✅ 메시지 읽음 처리
- ✅ 메시지 타임스탬프 표시
- ✅ 메시지 시간 포맷팅 (방금 전, N분 전, N시간 전, 날짜)
- ✅ 채팅 내역 스크롤 자동 이동

### 채팅 시작
- ✅ 사용자 목록에서 채팅 시작 (50포인트 차감)
- ✅ 게시글에서 채팅 시작 (50포인트 차감)
- ✅ 포인트 부족 시 알림

## 📝 3. 게시글 기능 (별톡)

### 게시글 작성
- ✅ 텍스트 게시글 작성 (최대 500자)
- ✅ 이미지 첨부 (1장)
- ✅ 갤러리에서 이미지 선택
- ✅ 카메라로 사진 촬영
- ✅ 이미지 미리보기 및 삭제
- ✅ 게시글 등록 시 Firestore 저장

### 게시글 조회
- ✅ 게시글 목록 조회 (최신순 정렬)
- ✅ 게시글 이미지 확대 보기
- ✅ 조회수 표시
- ✅ 게시글 작성자 정보 표시 (이름, 나이, 성별)
- ✅ 게시글 작성 시간 표시

### 게시글 필터링
- ✅ 지역별 필터링 (전체/17개 시도)
- ✅ 내 게시글만 보기
- ✅ 성별 필터링 (전체/남/여)
- ✅ BDSM 성향 필터링 (전체/지배/복종/스위치/없음)
- ✅ 필터 조합 가능
