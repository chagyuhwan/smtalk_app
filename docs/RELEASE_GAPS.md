# 실제 앱 출시 시 부족한 점

## 🔴 최우선 (출시 전 필수)

### 1. 법적 요구사항

#### 1.1 성인인증 시스템 ⚠️ **가장 중요**
- [ ] **본인인증 연동 필수** (NICE평가정보, KG모빌리언스 등)
  - **현재 상태**: 전화번호 인증만 있음 (나이 입력만으로 검증)
  - **문제점**: 사용자가 거짓 나이를 입력할 수 있음
  - **필요 작업**:
    - 본인인증 API 연동 (NICE평가정보 또는 KG모빌리언스)
    - 만 19세 이상 검증 (본인인증 서버에서 확인)
    - 성인인증 완료 여부 Firestore에 저장
    - 미성년자 가입 시도 차단 및 안내
  - **예상 비용**: 월 사용료 (건당 과금 또는 월 정액)
  - **예상 시간**: 3-5일

#### 1.2 이용약관 및 개인정보 처리방침 완성 ⚠️ **법무 검토 필수**
- [ ] **이용약관 완성**
  - **현재 상태**: 템플릿 상태 (`[회사명]`, `[서비스명]`, `[시행일]` 등 플레이스홀더)
  - **필요 작업**:
    - 실제 회사명으로 교체
    - 실제 서비스명으로 교체
    - 실제 시행일자 명시
    - 실제 연락처 정보 입력
    - **변호사 법무 검토** (필수)
  - **예상 비용**: 변호사 상담비 50-100만원
  - **예상 시간**: 2-3일 (법무 검토 포함)

- [ ] **개인정보 처리방침 완성**
  - **현재 상태**: 템플릿 상태
  - **필요 작업**:
    - 실제 회사 정보로 교체
    - 수집 항목 명확화 (전화번호, 이름, 나이, 성별, 위치정보 등)
    - 보관 기간 명시
    - 제3자 제공 정보 명시 (Firebase, 결제 서비스 등)
    - **변호사 법무 검토** (필수)
  - **예상 비용**: 변호사 상담비 포함
  - **예상 시간**: 2-3일 (법무 검토 포함)

- [x] **약관 동의 저장 기능** ✅ **구현 완료**
  - `setUserAgreement` 함수로 Firestore에 저장됨
  - 회원가입 시 동의 내역 저장됨

#### 1.3 개인정보 처리방침 URL
- [ ] **웹사이트 호스팅 필요**
  - 앱스토어/플레이스토어에 등록할 URL 필요
  - 웹사이트에 개인정보 처리방침 페이지 호스팅
  - **예상 비용**: 웹사이트 호스팅 월 1-5만원
  - **예상 시간**: 1일
