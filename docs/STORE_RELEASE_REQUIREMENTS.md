- [ ] **이용약관 완성**

  - 필요: 회사명, 서비스명 등 실제 정보로 교체
  - 시행일자 명시
  - **법무 검토 완료** (변호사 상담 필수)
- [ ] **개인정보 처리방침 완성**
  - 현재: 템플릿 상태
  - 필요: 
    - 수집 항목 명확화
    - 보관 기간 명시
    - 제3자 제공 정보 명시
    - **법무 검토 완료** (변호사 상담 필수)
- [ ] **회원가입 시 약관 동의 필수**
- [ ] **약관 동의 내역 저장** (Firestore)

#### 1.3 개인정보 보호
