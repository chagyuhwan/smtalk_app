### 3. 성능 테스트

#### 3.1 앱 시작 시간
- [ ] **콜드 스타트** (앱 완전 종료 후 시작)
  - 목표: 3초 이내
  - 측정 방법: 앱 아이콘 탭부터 첫 화면 표시까지

- [ ] **웜 스타트** (백그라운드에서 복귀)
  - 목표: 1초 이내
  - 측정 방법: 앱 전환부터 화면 표시까지

#### 3.2 메모리 사용량
- [ ] **메모리 사용량 확인**
  - iOS: Xcode Instruments > Allocations
  - Android: Android Studio Profiler > Memory
  - 목표: 100MB 이하 (일반 사용 시)

- [ ] **메모리 누수 확인**
  - [ ] 장시간 사용 후 메모리 증가 확인
  - [ ] 화면 전환 후 메모리 해제 확인
  - [ ] 이미지 로딩 후 메모리 해제 확인

#### 3.3 네트워크 성능
- [ ] **Firestore 쿼리 속도**
  - [ ] 사용자 목록 로딩: 2초 이내
  - [ ] 게시글 목록 로딩: 2초 이내
  - [ ] 메시지 로딩: 1초 이내

- [ ] **이미지 로딩 속도**
  - [ ] 프로필 이미지: 1초 이내
  - [ ] 게시글 이미지: 2초 이내

#### 3.4 배터리 사용량
- [ ] **배터리 소모 확인**
  - [ ] 백그라운드 실행 시 배터리 소모 확인
  - [ ] 실시간 리스너가 배터리를 과도하게 소모하지 않는지 확인

#### 3.5 렌더링 성능
- [ ] **스크롤 성능**
  - [ ] 사용자 목록 스크롤: 60fps 유지
  - [ ] 게시글 목록 스크롤: 60fps 유지
  - [ ] 채팅 메시지 스크롤: 60fps 유지

- [ ] **애니메이션 성능**
  - [ ] 화면 전환 애니메이션: 부드러움 확인
  - [ ] 로딩 애니메이션: 부드러움 확인
